# How to create github token

To create a GitHub token, follow these steps:

1. Log in to your GitHub account.
2. Click on your profile picture in the top right corner and select "Settings" from the dropdown menu.
3. In the left sidebar, click on "Developer settings."
4. Click on "Personal access tokens" in the left sidebar.
5. Click the "Generate new token" button.
6. Give your token a descriptive name in the "Note" field (e.g., "K8sResourceResizer Token").
7. Select the appropriate scopes for your token. For this project, you may need to select "repo" (for full control of private repositories) and "workflow" (to access GitHub Actions).
8. Click the "Generate token" button at the bottom of the page.
9. Copy the generated token and store it securely, as you won't be able to see it again. You will need this token to authenticate with the GitHub API when creating pull requests or performing other actions.
